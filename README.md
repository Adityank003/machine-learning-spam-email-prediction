# machine-learning-spam-email-prediction
Machine learning - Classifying emails into spam or not spam using Python and sci-kit learn.

Requirements:

1.Python 2.7+

2.Sci-kit learn

3.Pandas

4.Email data set

You can get it here https://archive.ics.uci.edu/ml/datasets/spambase
